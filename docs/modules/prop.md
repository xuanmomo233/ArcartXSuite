# Prop 快捷道具

## 功能定位

把物品标记为"快捷道具"，绑定 ArcartX 客户端按键效果和临时属性加成。

## 依赖

- 必需：ArcartX
- 可选：MythicLib / AttributePlus（属性加成）

## 启用步骤

```yaml
modules:
  prop:
    enabled: true
    password: "AXS-Prop@2026#Ready"
```

## 命令

```
/AXS prop status
/AXS prop reload
/AXS prop set <propId>     # 把手持物品标记为指定 prop
```
